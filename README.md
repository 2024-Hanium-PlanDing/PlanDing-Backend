# PlanDing-Backend
Planning을 주제로한 프로젝트

### 소개
---

PlanDing은 일정 관리와 그룹 스터디 플래너의 경계를 넘어서는 플랫폼입니다.
사용자의 일상과 학습 활동을 지원하기 위해 설계된 PlanDing은 직관적인 인터페이스와 함께, 일정 계획부터 실행까지 일련의 과정을 단순화하고 향상시키는 다양한 기능을 제공합니다.
